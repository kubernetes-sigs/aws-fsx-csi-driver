# v0.10.1

### Announcement
To improve the security of the container images, the base image will be substantially reduced in scope to only contain the necessary driver dependencies in an upcoming release.
**The CSI driver image should only be used in the CSI driver Deployment and Daemonset pods as documented in our Helm chart and Kustomize manifests. While this change wonâ€™t negatively impact workloads that rely on volumes managed by the FSx for Lustre CSI Driver, it may break unsupported uses of the CSI driver image outside the aforementioned official deployment methods.**
